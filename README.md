# 👨🏽‍💻 Desenvolvedor & QA

Me chamo Rafael Santos e sou formado em Análise e Desenvolvimento de Sistemas pela Universidade Católica de Brasília. Sou apaixonado pela área de Desenvolvimento de Software e Quality Assurance (QA.)

Tenho experiência e conhecimento em CSS, JavaScript e Python, além de certificações como ISTQB e CTFL, que reforçam meu compromisso com qualidade e testes de software. Recentemente, conquistei um certificado em Cypress para Web, API e GitHub Actions, o que me permitiu aprimorar ainda mais minhas habilidades em automação de testes.
